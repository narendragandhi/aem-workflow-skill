## Contributing

We welcome contributions! Please see [CONTRIBUTING.md](./CONTRIBUTING.md) for guidelines.

## Recognition

Contributors are recognized in the following ways:

1. **CONTRIBUTORS.md listing** - All contributors with documented contributions
2. **Release notes** - Significant contributions mentioned in changelog
3. **Documentation credit** - Major documentation contributions credited inline

## How to Get Listed

To be listed as a contributor:

1. **Code contributions** - Submit a pull request that is merged
2. **Documentation** - Improve docs with substantial additions or corrections
3. **Testing** - Provide and maintain comprehensive test coverage
4. **Bug reports** - Submit reproducible bug reports with fixes
5. **Examples** - Provide production-ready examples and patterns
